    Views maps as as map things as long as keys and values are viewed as things too. This only is well-defined if the equality of keys coincides with equality of Anything on the domain of keys. This is for example the case if keys is Anything, [[NumberT]], [[IntT]], [[NatT]] or [[StringT]]. If you cannot guarantee for this to be the case, use HashMaps instead.
