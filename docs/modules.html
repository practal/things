MapT

  • MapT<Key, Value>(keyT: Thing<Key>, valueT: Thing<Value>): MapThing<Map<Key, Value>, Key, Value>
    • Defined in map.mts:18

    Views a Map as as map thing as long as keys and values are viewed as things too. This only is well-defined if the equality of keys coincides with equality of Anything on the domain of keyT. This is for example the case if keyT is Anything, NumberT, IntT, NatT or StringT. If you cannot guarantee for this to be the case, use HashMapT instead.
